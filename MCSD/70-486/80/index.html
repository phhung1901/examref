Multiple Choices

What are compelling reasons to switch from static web content to dynamic web content? (Choose all that apply.)

  • A. The ability to substitute a new image for the company logo
  • B. The ability to display information from a database
  • C. The ability to link to other pages outside of your application
  • D. The ability to display information pertaining to the current user

Answer:

BD
Explanation
A. Incorrect: There is no need to create a dynamic website simply to replace the company logo.
B. Correct: Displaying data from a database is an ideal reason to use a dynamic website.
C. Incorrect: Linking to other pages outside of an application does not require a dynamic website.
D. Correct: Personalization, or displaying information about the user on-screen, is an example of dynamic content.
