You are developing an ASP.NET MVC application that allows users to log on by using a third-party authenticator.

You need to configure Microsoft Azure Access Control Services and the application.

Which five actions should you perform in sequence?

To answer, move the appropriate actions from the list of actions to the answer area and arrange them in the correct order.

  • Generate provider rules for claims.
  • Register the application as the relying party.
  • Add a Security Token Services (STS) reference in Visual Studio 2012
  • Create a service namespace
  • Add the third-party as the identity provider.
  • Add a symmetric key service identity.

            Answer:

            • Add a symmetric key service identity.
            • Create a service namespace
            • Register the application as the relying party.
            • Add a Security Token Services (STS) reference in Visual Studio 2012
            • Add the third-party as the identity provider.
            • Generate provider rules for claims.
